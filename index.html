DA Chat: AI Educational Assistant

A comprehensive AI assistant designed to help De Anza College students navigate campus resources and academic transfer processes. The system utilizes Retrieval-Augmented Generation (RAG) to answer queries regarding college policies and procedures with high accuracy.

It features a gamified transfer planning module and an intelligent course counselor agent that utilizes OpenAI function calling to query a backend MySQL database for real-time course availability. Additionally, the platform includes a multimodal "ChatPDF" module, allowing students to build personal knowledge bases from academic documents.

Spring Boot Vue.js OpenAI API MySQL

NLP Pipeline for Medical Data Structuring

Clinical Note Structure Extraction

Designed a high-precision NLP pipeline to convert unstructured clinical notes into standardized, structured outputs. The architecture leverages Low-Rank Adaptation (LoRA) to efficiently fine-tune Large Language Models (LLMs) for medical context, integrated with MedSpacy for domain-specific entity recognition. To ensure State-of-the-Art (SOTA) accuracy, the system employs Pydantic for strict output schema validation and implements an "Agent as Judge" evaluation framework to iteratively refine extraction quality.

Python PyTorch MedSpacy LoRA
